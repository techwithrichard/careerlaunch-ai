# Phase 03 · Track 02 — Job Search Workflows

Deliver end-to-end tooling for learners to discover, track, and manage opportunities.

## A. Job Aggregation
- [ ] Configure ingestion adapters for LinkedIn, Indeed, Glassdoor, and niche boards.
- [ ] Normalize job data (titles, skills, salary) for consistent filtering.
- [ ] Implement deduplication logic across sources and history.
- [ ] Support per-school filters (preferred employers, restricted industries).

## B. Discovery Experience
- [ ] Build advanced search with facets (location, compensation, remote, visa).
- [ ] Offer AI-powered job recommendations based on profile and activity.
- [ ] Enable saved searches and alerts with frequency controls.
- [ ] Provide job comparison view with pros/cons and mentor insights.

## C. Application Tracking
- [ ] Create Kanban board for pipeline stages with drag-and-drop.
- [ ] Allow manual and automatic creation of application entries.
- [ ] Attach resumes, cover letters, and interview notes to each record.
- [ ] Integrate tasks/subtasks (follow-ups, thank-you notes) with reminders.

## D. Calendar & Coordination
- [ ] Sync interviews to Google/Outlook calendars with timezone handling.
- [ ] Offer availability polling for mentor prep sessions.
- [ ] Include travel checklist for in-person interviews.
- [ ] Surface conflicts or overlaps with other scheduled events.

## E. Notifications & Nudges
- [ ] Send multi-channel alerts (email, push, in-app) for status changes.
- [ ] Provide weekly digest summarizing pipeline health.
- [ ] Notify mentors of learner milestones needing feedback.
- [ ] Log notification delivery and engagement for tuning.
