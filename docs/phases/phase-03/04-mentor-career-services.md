# Phase 03 · Track 04 — Mentor & Career Services Tools

Equip mentors and career staff with actionable insights and collaboration tools.

## A. Mentor Dashboard
- [ ] Surface cohort health metrics (applications, interviews, offers).
- [ ] Highlight learners needing attention based on inactivity or flagged risks.
- [ ] Provide quick actions (schedule session, send reminder).
- [ ] Embed analytics widgets from transparency service.

## B. Feedback Workflows
- [ ] Support structured feedback templates for resumes, interviews, networking.
- [ ] Allow mentors to assign action items with due dates.
- [ ] Track learner acknowledgment and completion of feedback tasks.
- [ ] Maintain feedback history for compliance and quality assurance.

## C. Communication Channels
- [ ] Implement secure messaging with thread context per learner.
- [ ] Enable mentor notes visible to staff but not learners (where appropriate).
- [ ] Integrate virtual meeting links (Zoom, Teams) with scheduling workflows.
- [ ] Provide shared resource library (articles, workshops) tagged per topic.

## D. Career Services Administration
- [ ] Manage employer partner database with outreach tracking.
- [ ] Publish upcoming events (career fairs, workshops) with registration.
- [ ] Generate reports summarizing placement outcomes per cohort.
- [ ] Handle mentor staffing levels and availability planning.
